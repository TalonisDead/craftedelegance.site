== Description ==

**Schedule upcoming sales in your WooCommerce store and even specify TIMES for the sale to begin and end!**<br>

1. Pick the begin and end of the sale - you can even specify the hour!
1. Specify the % discount off regular price.
1. Give it a name for your reference only.

Simply the FASTEST way to schedule sales in WooCommerce!

Compatible with WooCommerce 2.6 and higher, including 3.0+.

> **The <a href="https://pimwick.com/pw-woocommerce-on-sale">Pro version</a> lets you specify a category of products that are on sale!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pw-woocommerce-on-sale` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to Pimwick Plugins -> PW On Sale!

== Frequently Asked Questions ==
**Can I schedule a promotion that doesn't end until I stop it?**

* Yes, just set the End Date far into the future and when you're ready to end the promotion, just delete it or set the End Date to the past.
